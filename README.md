# Auditor Web - Ultimate Red Team Edition
Ferramenta de auditoria de segurança desenvolvida em Python para análise de infraestrutura e vulnerabilidades web.

## Funcionalidades:
- **Sessão 0:** Calibragem de Falsos Positivos.
- **Sessão 1/2:** Reconhecimento DNS e Geo-IP.
- **Sessão 4:** Auditoria de Cabeçalhos de Segurança (OWASP).
- **Sessão 5/6/7:** Motor de Fuzzing Paralelo (15 Threads - Modo Industrial).
- **Sessão 8:** Varredura de Portas Operacionais.
